# Project Templates

Pre-built agent + skill bundles for common project types. These serve as starting points — adjust based on the specific project's needs.

---

## Robotics Project

**Use when**: ROS2-based robot development, sensor integration, autonomous navigation, manipulation.

**Agents**:
- cto-orchestrator, project-manager, docs-generator (always)
- robotics-engineer, computer-vision-engineer, rl-engineer
- embedded-firmware-engineer, edge-computing-architect
- qa-test-engineer, devops-infra-engineer

**Skills** (beyond universal 5):
- ros2-development, sensor-fusion-pipeline, motion-planning-control, sim-to-real-transfer
- pytorch-training-pipeline, detection-segmentation-pipeline, 3d-perception
- rl-algorithm-implementation, reward-engineering, environment-design
- edge-deployment, containerization-orchestration, test-pyramid
