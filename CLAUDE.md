The main purpose of this project to write skills/agents that can be used for:
- Technology Research
- Software Development
- Academic Research
- Technology Entrepreneurship


Especially for research skills, in general principle should be finding actually meaningful, truthful and reliable sources. For instance, if user asks for researching topic in X (usually this is computer science, and more specificially DL/CV/Robotics topic, you should start research by looking papers from top conference, journal papers like CVPR, ICCV, ECCV, ICRA, RSS, ICRA, SIGGRAPH, ICML, ICLR, NeurIPS etc.)

The human whom will use these AI agents have master's degree in computer science and knowledgable in
- Self-supervised Learning
- Perception in Robot Learning (specificially 2D/3D perception)
- JEPAs
- World Models (at a basic level, not in depth of research)
